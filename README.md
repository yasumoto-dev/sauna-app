## 🚩概要

サウナ施設の情報を登録・閲覧し、各施設に対してレビューを投稿できるWebアプリです。
FlaskとSQLAlchemyを用いて開発し、Renderにデプロイしています。
施設一覧表示、施設登録、詳細表示、レビュー投稿を実装しました。

---

## 🎯主な機能

- サウナ施設の一覧表示
- サウナ施設の新規登録
- 施設ごとの詳細表示
- 施設ごとのレビュー投稿
- 同じ名前の施設の重複登録防止

---

## ⚙️使用技術

---

## 🦿工夫した点

- 施設一覧、施設詳細、レビュー投稿までの流れが直感的にわかるよう、シンプルな画面構成にしました
- 同じ名前の施設が重複登録されないように、チェック処理を実装しました
- Renderにデプロイした際、SQLiteのテーブルや初期データがそのままでは反映されなかったため、初回起動時に最低限の施設データを投入する処理を追加し、公開時に空画面にならないように工夫しました
